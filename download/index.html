Download

kalk is supported on the following platforms:

  • Windows x64
  • Linux x64 (Debian derivatives - e.g Ubuntu and Redhat)
  • macOS

The current stable version: 0.1.0

Windows

The easiest and preferred way to install kalk is from the Windows Store. This version can be updated automatically.

The application is being published and is not yet available via the Windows Store.

Alternatively, you can download the zip archive kalk.0.1.0.win-x64.zip

Linux

On Ubuntu and all Debian derivatives

$ wget https://github.com/xoofx/kalk/releases/download/0.1.0/kalk.0.1.0.linux-x64.deb
$ sudo apt install ./kalk.0.1.0.linux-x64.deb

On CentOS, RHEL & Fedora (RPM)

$ wget https://github.com/xoofx/kalk/releases/download/0.1.0/kalk.0.1.0.linux-x64.rpm

macOS

You can install kalk with homebrew

$ brew tap xoofx/kalk 
$ brew install kalk

You can also download a tar.gz archive kalk.0.1.0.osx-x64.tar.gz

Installation with .NET

If you have .NET 5.0 SDK installed, kalk is available on NuGet a .NET global tool:

$ dotnet tool install --global kalk --version 0.1.0
